# Student-Performance-Analysis-Excel-Pivot-Table-Project

## 📌 Project Overview
This project demonstrates the use of Excel Pivot Tables to analyze and summarize student performance data. The goal was to transform a raw dataset of student scores into meaningful insights regarding gender-based performance, house distributions, and age demographics.

## 📊 Dataset Description
The dataset consists of student records including:
Demographics: Name, Gender, Age, Class, and House (Agni, Bhoomi, Jal, Vayu).
Academic Scores: Unit Test 1, Unit Test 2, and Final Test scores.

## 🛠️ Skills Demonstrated
Data Cleaning: Preparing raw Excel data for analysis.
Pivot Tables: Aggregating data to find sums, averages, and counts.
Data Filtering: Using Slicers/Filters to drill down into specific classes and age groups.
Multi-level Grouping: Nested row labels (House > Gender > Age) for granular reporting.

## 🔍 Key Insights & Analysis
1. Gender Performance Gap
Analyzed the total scores in the Final Test to determine which gender performed better overall.
Observation: Females scored significantly higher in total (1090) compared to males (877).

2. House-wise Performance by Gender
A deeper dive into how different "Houses" performed, broken down by gender.
Finding: In the Vayu house, females outperformed males by a wide margin (460 vs 186), whereas in the Agni house, males had a higher cumulative score.

3. Advanced Filtering (Class & Age)
Utilized report filters to summarize Unit Tests and Final Test averages.
Functionality: This allows users to toggle between specific classes (e.g., Class 10 vs Class 8) to see how academic trends shift as students get older.

4. Demographic Summary
Created a count-based summary to understand the student distribution.
Structure: House → Gender → Age → Count.
Insight: This helps identify the "density" of students in each category, ensuring the averages in previous steps aren't skewed by a single outlier.
